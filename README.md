
# Proyecto de Vulnerabilidades Web - OWASP Top 10

Este proyecto tiene como objetivo enseñar y demostrar vulnerabilidades web comunes, basadas en el OWASP Top 10. Aquí encontrarás ejemplos prácticos, código educativo y guías para identificar, explotar y mitigar riesgos en aplicaciones web.

## Características
- Ejemplos de vulnerabilidades OWASP Top 10
- Código comentado y educativo
- Guía para pruebas de seguridad
- Recomendaciones de mitigación

## Instalación

Clona el repositorio:
```bash
git clone <url-del-repositorio>
```
Cada laboratorio tendrá sus propias instrucciones para levantar el entorno.

## Uso

Vulnerabilidades implementadas:
- IDOR
- User Enumeration

Próximamente:
- Broken access control - escalamiento horizontal
